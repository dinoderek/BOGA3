# Mobile App (Milestone 0 Smoke)

## Local startup

1. Install dependencies:

```bash
npm install
```

2. Start Expo (offline first):

```bash
npx expo start --offline
```

3. If offline mode is unavailable in your environment, use:

```bash
npx expo start
```

4. Open in Expo Go or simulator and verify `/` shows:

`Milestone 0 foundation ready`
